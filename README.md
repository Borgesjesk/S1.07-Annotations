# S1.07 - Annotations

## 📌 Exercise Description
Deep dive into Java annotations: built-in annotations like @Override and @Deprecated, and custom annotations with Reflection to influence program behaviour at runtime.

## ✨ Functionalities

### Level 1 - Built-in Annotations
- **Exercise 1:** Worker hierarchy demonstrating @Override with polymorphic salary calculation
- **Exercise 2:** @Deprecated methods in child classes, invoked with @SuppressWarnings from external class

### Level 2 - Custom Annotations
- **Exercise 1:** Custom @JsonSerializable annotation with Reflection-based JSON serializer, writing annotated objects to .json files

## 🛠 Technologies
- Java 21
- Java Reflection API
- Java Annotations (built-in and custom)
- Maven 3.9.11
