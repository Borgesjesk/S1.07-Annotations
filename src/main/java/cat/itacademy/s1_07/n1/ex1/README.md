# N1 Exercise 1 - @Override

## 📌 Exercise Description
Create a Worker hierarchy with three classes demonstrating the @Override annotation through polymorphic salary calculation.

## ✨ Functionalities
- Worker base class with name, surname, pricePerHour and calculateSalary()
- OnlineWorker adds a fixed internet fee constant to the salary
- OnsiteWorker adds a static fuel allowance to the salary
- Both child classes override calculateSalary() using @Override
- Main demonstrates polymorphism via List<Worker>

## 🛠 Technologies
- Java 21
- @Override annotation
- Maven
