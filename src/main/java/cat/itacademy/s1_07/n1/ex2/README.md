# N1 Exercise 2 - @Deprecated

## 📌 Exercise Description
Add deprecated methods to the Worker child classes and invoke them from an external class using @SuppressWarnings.

## ✨ Functionalities
- getBonus() marked as @Deprecated in OnlineWorker and OnsiteWorker
- External Main class calls deprecated methods
- @SuppressWarnings("deprecation") suppresses compile-time warnings

## 🛠 Technologies
- Java 21
- @Deprecated annotation
- @SuppressWarnings annotation
- Maven
